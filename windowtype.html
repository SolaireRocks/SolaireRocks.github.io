Description　
▲　

In FF6 T-Edition I've changed the window types available. The default window is in Square's game style.
Also included are 3 patches of various windows you can patch onto the game to choose from. "Window 1" is other game styles, "Window 2" is NES game styles, and "Window 3" is original works. These are not made by me but others who have done an amazing job.

Please use the patches on an already patched ROM of FF6 T-Edition or EX. Nothing except the window choices are changed. This patch is only for FF6 T-Edition and EX. If used on a normal game it won't have any effect due to moving where the window graphic is stored in the memory.

※Some characters are difficult to see based on the window selected. This is a specification.

Sometimes the colors will be off after patching. Simply highlight the cursor on the "Colors" option in the Config menu at the window select and press the A Button to reset it to the default.

If you decide to change your windows and want to revert back to the 8 original ones T-Edition provides, you will need to reload a previous .srm file. It is strongly recommended in this case you make a back up file first before swapping windows over in the event you want them back later -Kain.
